## Features

* Track your cigarettes smoked in real time.
* View statistics and progress.
* Beautiful and smooth UI with modern animations.

## 🛠 Technologies used

The project is written in **100% Kotlin** using modern Android development practices:

* **[Jetpack Compose](https://developer.android.com/jetpack/compose)** — UI-instrumets.
* **[Material Design 3](https://m3.material.io/)** — design system and interface components.
* **[Navigation Compose](https://developer.android.com/jetpack/compose/nav-adaptive)** — navigation.
